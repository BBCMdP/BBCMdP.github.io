Proteomes Database

An interactive web database for browsing, filtering, and exporting annotated complete proteomes, with taxonomy-aware exploration, BUSCO quality metrics, and collection-based workflows.

Main capabilities

  • Advanced filtering by taxonomy, BUSCO completeness, and curated collections.
  • Configurable result columns, sorting, and TSV export for downstream analysis.
  • Lifemap integration for phylogenetic exploration of selected proteomes.
  • Responsive frontend backed by a materialized-table API for fast queries.

Current status

  • Publicly accessible from this site.
  • Connected to the dedicated Proteomes Database page and API backend.
  • Being kept online while the broader projects section is rebuilt.
