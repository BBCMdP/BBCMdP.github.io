How to use filters and explore data

expand_more
Search (text): Searches across multiple proteome fields: Hash, Proteome source, AssemblyID, URL, NCBI refseq, Species, taxID, taxID at species level, NCBI's Current Scientific Name, Common name, NCBI's Group name, Informal clade, and Proteome's code renamed.
Hash (exact): Enter one or more proteome hashes (comma, space, or newline separated) to find specific proteomes.
Preset collection: Pre-defined proteome datasets, typically associated with specific or broad taxonomical distributions.
Taxonomy filters: First select a Taxonomy Level, then choose a Taxon Name from the dropdown. Use these together to narrow by taxonomy.
BUSCO quality: Select a BUSCO metric and set a minimum value (≥) to filter proteomes by quality.
Column selection: Customize which columns appear in results. Changes take effect when you click "Apply Filters".
Export: Download results as TSV. Lifemap: Explore the taxonomical distribution of results using taxID at species level.
